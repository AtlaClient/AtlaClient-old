AtlaClient v0.9.1.11 (539Kb)
Клиент под Windows9x/NT для Атлантиса v4.0.4.

Основные возможности:
- хранение информации обо всех загруженных ходах,
- виртуальная отработка ходов,
- поддержка нескольких игр и нескольких фракций в одной игре.

Автор: Евгений Пашкин

Изменения:
0.9.1.11:
- в readme.txt добавлено описание файлов настроек.
- ВНИМАНИЕ: следующая версия не будет воспринимать файлы версий меньше
данной, для приведения файлов к текущей версии нужно выполнить команду
Fix/Repack Store.
- в диалог создания нового юнита добавлена возможность установки флагов.
- добавлена возможность сдвигать приказы вверх и вниз по списку.
- добавлена поддержка дорог (при наличии двух сообщающихся дорог
можно двигаться (не лететь) в два раза быстрее).
- добавлена поддержка предметов, помогающих делать другие предметы
(для dsnlab).
- добавлена новая команда NOP, убирающая сообщение "no monthlong orders".
- добавлены новые настройки в Settings\Options.
- исправлено чтение длинных приказов teach из отчетов
(спасибо Андрею Репину).
- добавлен экспорт карты.

0.9.1.10:
- добавлена блокировка на время выполнения хода.
- изменен импорт из AtlaCart'а: подземные регионы добавляются
только если подземный уровень уже присутствует, но у них не
добавляются выходы.
- добавлен возможность не сохранять игру при создании
новой игры и смене игры
- в контекстное меню для Информации добавлена команда
копирования строки в буфер обмена.
- в настройках добавлена возможность запрещать появление
закрытого окна сообщений "Show closed Events window" и
отключение запуска хода для региона при смене юниты и региона
"Run Orders on change", обрезка имени города на карте
"Town name limit".
- окна Settings\Skill types, Skill reports, Item types
сделаны немодальными.
- исправлена часть багов при загрузке неполных отчетов.
- при удалении приказа-сообщения ;received ... from ...
удаляется соответствующий приказ give.
- в Receive не показываются юниты без получаемого предмета.
- в Wedges добавлена функция presenlocaltitem(itemname) и
presennonlocaltitem(itemname), возвращающие количество предметов
данного типа у локальных и остальных юнитов.
